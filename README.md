# alt-ppt4lecture

Single-file slide editor with 16:9 fixed stage and custom `.altppt` packages.

Download
- HTML app: https://github.com/sykaboom/alt-ppt-lecture/raw/master/alt-ppt4lecture.html?download=1

Usage
- Open `alt-ppt4lecture.html` in Chrome or Edge.
- Load or save `.altppt` packages from the toolbar.
